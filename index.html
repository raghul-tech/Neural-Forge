🎯 WHAT IS THIS GAME?

Build a network of machines that learn over time. Each machine is like a brain cell. Connected together, they produce SCORE points.

Goal: Reach 1000 SCORE within 15 loops.

🎮 QUICK START (60 SECONDS)

1️⃣ Click SENSOR machine below
2️⃣ Click on the empty grid to place it
3️⃣ Place another SENSOR
4️⃣ Place 2 PROCESSORS
5️⃣ Place 1 ACTUATOR
6️⃣ Click EVOLVE LOOP → Watch your score grow!

💡 WINNING FORMULA

Loop 1-3: 2 SENSORS + 2 PROCESSORS + 1 ACTUATOR

Loop 4-7: Add 2 ACTUATORS + 1 MEMORY

Loop 8-12: Add CONNECTORS + MUTATOR

Loop 13-15: Keep adding ACTUATORS → WIN!

🕹️ CONTROLS

  • Left Click - Place machine
  • Right Click - Remove machine
  • EVOLVE LOOP - End round, earn resources
  • AUTO-BUILD - AI places machines for you
