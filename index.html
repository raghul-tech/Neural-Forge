🎯 WHAT IS THIS GAME?

Neural Forge is a machine-building strategy game. You build a network of machines that learn and improve over time. Each "loop" is like a round - your machines produce resources, and you use those resources to build more machines!

🏆 GOAL

Reach 1000 SCORE points within 15 loops to win!

🎮 HOW TO PLAY (STEP BY STEP)

1️⃣ SELECT A MACHINE - Click on any machine in the Machine Library below the grid
2️⃣ PLACE IT - Click anywhere on the empty grid to place your machine
3️⃣ REMOVE MACHINES - Right-click on any machine to remove it (get half resources back)
4️⃣ EVOLVE LOOP - Click "EVOLVE LOOP" to finish current round. Your machines will produce resources!
5️⃣ BUILD MORE - Use resources to build more machines in next loop
6️⃣ REPEAT - Keep building until you reach 1000 score!

🔧 MACHINE TYPES & WHAT THEY DO

MachineCostFunctionBest Used When
📡 SENSOR15Detects input, starts the chainStart of every network
🧠 PROCESSOR25Calculates signals, passes forwardMiddle of your network
💾 MEMORY30Stores past data, improves learningAfter you have 3+ machines
🦾 ACTUATOR20PRODUCES RESOURCES (your score!)End of your network
🔗 CONNECTOR10Links machines togetherWhen machines are far apart
🧬 MUTATOR40Evolves connections, boosts learningLate game, when rich

💡 BEGINNER STRATEGY (WIN IN 15 LOOPS!)

Loop 1-3: Build 2 SENSORS + 2 PROCESSORS + 1 ACTUATOR

Loop 4-7: Add MEMORY units and CONNECTORS

Loop 8-12: Add MUTATORS to boost learning

Loop 13-15: Keep adding ACTUATORS for maximum score!

🤖 AUTO-BUILD FEATURE

Click AUTO-BUILD and the AI will automatically place the best machine in an empty spot! Use this when you're not sure what to build next.

📊 WHAT THE NUMBERS MEAN

  • LOOP: Current round number (max 15)
  • INTELLIGENCE: How smart your network has become (higher = better production)
  • SCORE: Your progress toward 1000 goal 🏆
  • RESOURCES: Currency to buy machines (earned each loop)
  • Learning Rate: How fast machines adapt (higher = faster learning)

⌨️ QUICK CONTROLS

  • Left Click - Place selected machine
  • Right Click - Remove machine
  • EVOLVE LOOP - End round, gain resources
  • AUTO-BUILD - AI places machines for you
  • RESET - Start over
