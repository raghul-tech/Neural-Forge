# 📝 Changelog

## [1.0.0] - 2026-04-19 🚀

### 🎉 Initial Release - Gamedev.js Jam 2026

**Live Demo:** https://raghul-tech.github.io/Neural-Forge

### Added

#### Gameplay
- 6 machine types (Sensor, Processor, Memory, Actuator, Connector, Mutator)
- 15-loop progression system
- Target: 1000 score to win
- Resource management (start with 100)
- Intelligence & learning rate tracking
- Auto-build AI assistance

#### Features
- Click + drag to pan camera
- Right-click to remove machines
- Win/Loss popups with confetti
- Sound effects for all actions
- Game saves automatically
- Help modal with instructions

#### Technical
- Pure JavaScript (no frameworks)
- HTML5 Canvas rendering
- LocalStorage for saves
- Web Audio API for sounds
